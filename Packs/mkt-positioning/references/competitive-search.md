# Competitive Messaging Search

How to research competitor messaging before generating angles. This grounds positioning in market reality rather than assumptions.

---

## Search Process

### 1. Identify Search Targets

- If `brand_context/icp.md` mentions competitors, start with those
- If the user named competitors, search those
- Otherwise, search for "[product category] + [target market]" to find the top players
- Aim for 3-5 competitors minimum

### 2. Pull Messaging Data

For each competitor site, extract:

- **Homepage headlines and hero copy** — their primary positioning claim
- **Taglines and value propositions** — how they frame their offer
- **Key claims on feature/pricing pages** — what they believe differentiates them
- **Social proof framing** — how they present testimonials (outcome-focused? volume-focused? authority-focused?)
- **CTA language** — what action they want and how they ask for it
- **About page** — how they describe their origin story and mission

### 3. Map the Landscape

Categorise every claim into three buckets:

**Saturated territory** — claims that appear on 3+ competitor sites. These are dead. Using them won't differentiate.

**Partially claimed** — angles only 1 competitor uses. These may be effective but will trigger direct comparison.

**White space** — angles no competitor uses. This is where opportunity lives. An angle in white space has no competition for attention.

Also note:
- What proof/mechanism language dominates the space
- Whether competitors lead with features, outcomes, or identity
- Common CTA patterns (free trial, demo, download)

### 4. Present as Landscape Map

Format the findings clearly before generating angles:

```
Competitors Analyzed
- [Name] — "[their headline]"
- [Name] — "[their headline]"
- [Name] — "[their headline]"

Saturated Claims (everyone says this — avoid)
- [Claim 1]
- [Claim 2]
- [Claim 3]

Partially Claimed (1-2 competitors)
- [Claim] — used by [Competitor]
- [Claim] — used by [Competitor]

White Space (nobody is talking about this)
- [Gap 1 — what's missing]
- [Gap 2 — what audience needs but nobody addresses]
- [Gap 3 — underserved segment or angle]
```

---

## What to Do With the Map

- **Saturated claims** → Do not use these as primary angles. If your product genuinely delivers on a saturated claim, it can be supporting evidence but not the headline.
- **Partially claimed** → Usable if you can do it better or more specifically. Check if the competitor owning this claim is strong or weak.
- **White space** → Primary angle candidates. The best positioning occupies territory nobody else has claimed.

---

## When Web Search Is Unavailable

If web tools aren't available in the current environment:

1. Ask the user to name 3-5 competitors
2. Ask what claims those competitors make on their homepages
3. Ask what the user thinks everyone in the space says that sounds the same
4. Build a landscape map from their answers — less precise but still valuable

Note in the output: "Landscape built from your knowledge — a web search would make this more precise."
