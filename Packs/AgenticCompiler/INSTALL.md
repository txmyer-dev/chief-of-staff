# INSTALL: Agentic Compiler

This document guides the AI through installing the Agentic Compiler pack.

## Phase 1: System Analysis
1. Verify `node` is installed on the system.
2. Verify the `Packs/` directory exists in the current project.

## Phase 2: Backup
1. No critical files are overwritten by this pack.

## Phase 3: Installation
1. Create `Packs/AgenticCompiler/src/Tools/` directory.
2. Copy `src/SKILL.md` to `Packs/AgenticCompiler/src/SKILL.md`.
3. Copy `src/Tools/scaffold.js` to `Packs/AgenticCompiler/src/Tools/scaffold.js`.
4. Register `agentic-compiler` in `Packs/_catalog/catalog.json`.

## Phase 4: Verification
1. Run `VERIFY.md` to confirm installation.
